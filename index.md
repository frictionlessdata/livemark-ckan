# Livemark CKAN

It's a CKAN plugin for Livemark

## Installation

Install with a package manager:

```bash
pip install livemark-ckan
```

## Documentation

Enable the plugin:

> livemark.yaml

```yaml
ckan: true
```

Use it in documents:

```
'''yaml package/ckan
path: https://data.stadt-zuerich.ch/dataset/ted_taz_verkehrszaehlungen_werte_fussgaenger_velo
'''
```
